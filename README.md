# Java Installers

Installers that don't hide shit all over your system

---

## win-inst

**Usage:**

1. Run as admin

2. Enter path to jdk zip file

**Writes to:**

- C:\\Program Files\\Java

**Updated Environment Variables (System-wide)**

- JAVA\_HOME

- Path

**Registry Updates:**

- HKEY\_CLASSES\_ROOT\\.jar

- HKEY\_CLASSES\_ROOT\\.jar\\OpenWithProgids

- HKEY\_CLASSES\_ROOT\\jarfile
