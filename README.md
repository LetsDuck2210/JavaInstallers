# Java Installers

Installers that don't hide shit all over your system

---

## win-inst

**Usage:**
1. Run as admin
2. Enter path to jdk zip file

**Writes to:**
- C:\\Program Files\\Java
- %temp%

**Updated Environment Variables (System-wide)**
- JAVA_HOME
- Path

**Registry Updates:**
- HKEY_CLASSES_ROOT\\.jar
- HKEY_CLASSES_ROOT\\jarfile
